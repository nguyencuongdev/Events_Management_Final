## Description

Dự án được viết trong khoảng thời gian thi kỹ năng nghề cấp thành phố

## Run Project

php artisan migrate
php artisan serve

## ENVIROMENTS:

-- DB_HOST = server name database
-- DB_PORT = database listening port
-- DB_DATABASE =hanoidb
-- DB_USERNAME = username login database
-- DB_PASSWORD = password login database
